# 🇰🇷 한국건설기술연구원 (KICT) – 위성 SAR 기반 인프라 현황 모니터링

이 저장소는 [KICT](https://www.kict.re.kr)에서 공개한 **위성 SAR(Synthetic Aperture Radar)** 데이터를 활용한 **인프라 모니터링 기술** 관련 내용을 문서화한 것입니다.  
최신 연구 결과를 투명하게 공유하며, **누구나 기여**할 수 있습니다.
