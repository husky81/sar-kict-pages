# 소스코드 다운로드

윈도우에서 파워쉘에서 다음 명령 실행

```powershell
git clone https://github.com/husky81/sar-kict-pages
cd sar-kict-pages
npm install
npm run dev
```

git과 Node.js가 설치되어있지 않으면 다운 받아서 설치

다음 내용을 수정한 후 

문서 내용: .\src\content\docs\

문서 목록: .\astro.config.mjs
